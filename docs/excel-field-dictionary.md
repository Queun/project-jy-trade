# Excel 字段字典

本文件记录当前样例 Excel 的字段基线。字段固定时，系统应按这些列名读取和导出，不做列名猜测。

## 1. 订货单

文件：`ole案例文件——发货前\1订货单\订货通知单 .xls`

Sheet：`订货通知单`

行列规模：41 行，35 列。

表头行：第 1 行。

字段：

| 字段 | 用途 |
| --- | --- |
| 订货通知单号 | 订单明细归属单号 |
| 订货审批单号 | 上游审批单号 |
| 门店 | 客户门店编码 |
| 门店名称 | 客户门店名称 |
| 阅读状态 | 原始状态保留 |
| 送货方式 | 原始配送方式 |
| 状态 | 原始订单状态 |
| 送货地 | 门店/送货地显示值 |
| 大类 | 商品大类 |
| 订货日期 | 订单排序字段 |
| 截止日期 | 审核参考字段 |
| 上传时间 | 优先订单排序字段 |
| 业务员 | 审核参考字段 |
| 制单人 | 原始字段保留 |
| 制单时间 | 原始字段保留 |
| 审核人 | 原始字段保留 |
| 商品编码 | 甲方客户侧商品编码 |
| 商品名称 | 商品匹配辅助字段 |
| 商品条码 | 公开条码，优先用于匹配旺店通规格 |
| 规格 | 商品规格 |
| 运输规格 | 箱规/包装换算参考 |
| 未含税进价 | 价格参考 |
| 订货箱数 | 原始订货箱数 |
| 订货数 | 核心订货数量 |
| 折扣率 | 原始字段保留 |
| 保质期(天) | 审核参考字段 |
| 实收数量 | 原始字段保留 |
| 含税合同进价 | 价格参考 |
| 含税进价 | 价格参考 |
| 赠品率 | 原始字段保留 |
| TD | 原始字段保留 |
| DA | 原始字段保留 |
| PD | 原始字段保留 |
| SPD | 原始字段保留 |
| REBATE | 原始字段保留 |

当前样例导入结果：

- 明细行：40 行。
- 订单数：20 个。
- 门店数：20 个。
- 条码数：4 个。
- 重复条码数：2 个。

## 4. 旺店通批量做单模板

文件：`ole案例文件——发货前\4批量做单表格\批量导入模板.xls`

Sheet：`Sheet1`

字段共 49 列，核心字段：

| 字段 | 用途 |
| --- | --- |
| 店铺名称 | 固定或配置 |
| 原始单号 | 通常来自订货通知单号 |
| 收件人 | 地址匹配表派生 |
| 省 / 市 / 区 | 地址拆分或留空 |
| 手机 / 固话 | 地址匹配表派生 |
| 仓库 | 配置或审核结果 |
| 地址 | 地址匹配表派生 |
| 物流类别 | 固定或配置 |
| 仓库名称 | 做单目标仓 |
| 物流公司 | 做单物流 |
| 买家备注 / 客服备注 / 打印备注 | 按业务规则填充 |
| 发票类型 / 发票抬头 | 当前案例中存在默认值 |
| 业务员 | 原订货单业务员 |
| 商家编码 | 旺店通 `spec_no`，不是外部公开条码 |
| 商品数量 | 最终发货数量 |
| 商品价格 / 商品总价 / 商品优惠 | 按做单规则填充 |
| 平台 | 固定或配置 |

## 5. 地址匹配表

文件：`ole案例文件——发货前\4批量做单表格\地址匹配表格.xlsx`

当前文件包含多个历史 Sheet，表头不完全统一。优先使用结构最稳定的字段：

- 门店编号/群组。
- 门店名称。
- 门店地址。
- 收货人。
- 联系方式。

后续需要确定：

- 以哪个 Sheet 作为当前有效地址源。
- 门店编码是否唯一。
- 若同一门店多条地址，使用最新表、指定表，还是人工选择。
